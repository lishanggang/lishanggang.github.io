#SQL操作

##一.查询 SELECT
1.查询 用户表信息
SELECT * FROM user

3.确定数据表里有多少条数据记录(该表有多少个id)
SELECT COUNT(id) FROM tablename
4.查询数据表的某列的信息(如name列,age列,sex列)
SELECT name,age,sex FROM user

##二.WHERE子句

1.运算符

= 等于 >大于 >=大于等于 <小于 <=小于等于 <>不等于

###2.逻辑运算符
AND OR NOT 如果条件是FALSE ,返回TRUE
SELECT * FROM user WHERE name=b2;

##三. 限制查询结果中的数据记录个数(LIMIT)
LIMIT子句用于强制SELECT语句返回指定的记录数。
LIMIT接受一个或两个数字参数。参数必须是一个整数常量。
如果给定两个参数，第一个参数指定第一个返回记录行的偏移量，第二个参数指定返回记录行的最大数目。
注意：初始记录行的偏移量是0而不是1。

##四. 对查询结果进行排序 (ORDER BY)
使用ORDER BY子句对查询返回的结果排序。
ORDER BY子句的语法格式为： ORDER BY {column_name [ASC| DESC]} [,…n]
其中ASC表示升序，为默认值，DESC为降序 

例：
SELECT * FROM user ORDER BY uid  ASC  顺序
SELECT * FROM user ORDER BY uid  DESC 降序
SELECT * FROM user ORDER BY regdate DESC,username ASC;
SELECT * FROM user ORDER BY name DESC

##五. 插入数据记录(INSERT)
例：
INSERT INTO user (username,password) VALUES (‘admin’,’123456’)

INSERT INTO user (name,pwd) VALUES  ('chenxiao','923842')

INSERT INTO user (`id`,`name`,`pwd`) VALUES (NULL,'admin','123456')
一次插入多条记录
INSERT INTO user (name,pwd) VALUES ('chenxiao','13243'),('sdha','12433')

六.更新修改数据

例：
UPDATE user
SET username = 'admin1', passwd ='12345678'  WHERE id = 10

UPDATE user SET name='saha',pwd='adsfasdf' WHERE id=10

删除某一行

DELETE FROM user WHERE uid = 10
